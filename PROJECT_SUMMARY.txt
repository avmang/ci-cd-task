GETTING STARTED
================================================================================

1. Local Testing (5 minutes):
   docker build -t flask-app:local .
   docker run -p 8080:8080 flask-app:local
   curl http://localhost:8080/

2. GCP Setup (15 minutes):
   cd terraform
   cp environments/production.tfvars.example environments/production.tfvars
   # Edit with your values
   terraform init
   terraform apply -var-file=environments/production.tfvars

3. GitHub Configuration (5 minutes):
   - Add GitHub Secrets (from terraform outputs):
     * GCP_PROJECT_ID
     * GCP_WORKLOAD_IDENTITY_PROVIDER
     * GCP_SERVICE_ACCOUNT
   - Create environments: test, staging, production

4. Deploy:
   git push origin main
   # Watch GitHub Actions for deployment

CI/CD PIPELINE FLOW
================================================================================

Trigger: Push to main/develop or Pull Request
   │
   ├─> [Test Job]
   │   ├─ Checkout code
   │   ├─ Setup Python 3.11 (with pip cache)
   │   ├─ Install dependencies
   │   ├─ Run pytest with coverage
   │   └─ Upload coverage report
   │
   ├─> [Build & Scan Job] (after Test)
   │   ├─ Checkout code
   │   ├─ Setup Docker BuildKit
   │   ├─ Build image (with cache)
   │   ├─ Run Trivy security scan
   │   ├─ Upload SARIF to GitHub Security
   │   └─ Archive JSON report
   │
   ├─> [Deploy Staging] (if develop branch)
   │   ├─ Authenticate to GCP
   │   ├─ Build and push to Artifact Registry
   │   ├─ Deploy to Cloud Run (staging)
   │   ├─ Health check
   │   └─ Output deployment URL
   │
   ├─> [Deploy Production] (if main branch)
   │   ├─ Authenticate to GCP
   │   ├─ Build and push to Artifact Registry
   │   ├─ Deploy to Cloud Run (production)
   │   ├─ Health check
   │   └─ Output deployment URL
   │
   └─> [Notify Job] (always runs)
       ├─ Determine overall status
       ├─ Create job summary
       └─ Optional: Send Slack notification
